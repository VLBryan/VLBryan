# Proyectos destacados
- Segmentación de empresas en el uso del factoring en Perú (2024)  
    Este proyecto aplica técnicas de clustering y visualización interactiva para segmentar empresas proveedoras que utilizaron factoring en Perú durante 2024.
    
    ➡️ Repositorio https://github.com/VLBryan/Segmentaci-n-empresas-factoring-Per-.git

- Predicción de Enfermedad Cardíaca mediante Modelos de Machine Learning  
    Proyecto de ciencia de datos aplicado a salud: El objetivo fue implementar un modelo predictivo para la detección temprana de enfermedad cardíaca. Mi proyecto más representativo en cuanto a impacto y profundidad analítica.
    
    ➡️ Repositorio https://github.com/VLBryan/Detectar_Enfermedad_Cardiaca.git

- Dashboard de Monitoreo de Inseguridad Ciudadana  
    Proyecto académico con datos abiertos de la municipalidad de Miraflores. Se construyó un dashboard para visualizar y analizar tendencias de inseguridad.

- Dashboard de Ventas en Power BI  
    Integración de datos desde SQL y Excel, con medidas DAX para cálculos avanzados. Permite analizar tendencias de ventas y evaluar la rentabilidad de productos.

- Análisis Multivariado en R  
    Proyecto académico aplicado a datos de lectura en niños menores de 15 años. Se implementó un modelo de regresión múltiple para identificar variables relevantes.
